# Apply for Your Dream Role — Job Application Form

## Project Overview
A modern, elegant, fully responsive **Job Application Form** designed like a premium HR / careers portal. Built with **HTML5, CSS3, and Vanilla JavaScript**—no external validation libraries.

## Features
- Soft pastel **HR portal UI** with frosted card styling
- Accessible, semantic form layout (labels, aria-invalid, live error regions)
- Field-level validation:
  - Validate **on blur**
  - Validate **while typing** after the field is touched
  - Validate **on submit**
- Inline validation messages with soft red/green states
- Subtle **loading state** (1–2 seconds) before success
- Polished success **modal** with animated confirmation icon
- Live message character counter: **0/250**
- Optional form completion **progress indicator**
- Custom selection + scrollbar styling to match the theme

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- Google Font: **Inter**
- Font Awesome icons

## Validation Rules
- **Full Name**
  - Required
  - Min 3 characters
  - Alphabets and spaces only
- **Email Address**
  - Required
  - Must match a valid email format
- **Phone Number**
  - Required
  - Exactly 10 digits
  - Numbers only
- **Message**
  - Required
  - Min 20 characters
  - Max 250 characters
